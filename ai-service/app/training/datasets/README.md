# Defect Dataset

Use YOLO detection format under `defects/`.

```text
defects/
  images/
    train/
    val/
    test/
  labels/
    train/
    val/
    test/
```

Supported MVP classes:

```text
0 crack
1 damp
2 mould
3 peeling_paint
4 water_seepage
5 stain
6 wall_hole
7 tile_damage
8 poor_finish
```

Do not commit licensed or private dataset images unless the repository is allowed to store them.
